# Button

ButtonとLEDを連携します。

## ButtonとLEDの連携


``` python
# coding: utf-8
import RPi.GPIO as GPIO
import sys

LEDPIN = 4
BUTTONPIN = 5

GPIO.setmode(GPIO.BCM)
GPIO.setup(LEDPIN, GPIO.OUT)
GPIO.setup(BUTTONPIN, GPIO.IN)

try:
    while True:
        # ボタン押下判定
        if( GPIO.input(BUTTONPIN)):
            # LED点灯
            GPIO.output(LEDPIN, False)
        else:
            # LED消灯
            GPIO.output(LEDPIN, True)
except KeyboardInterrupt:
    GPIO.cleanup()
    sys.exit(0)
```
