# Dockerの保存と停止

JupyterLab経由ではなく、ssh経由でJetson Nanoにログインします。

```
ssh jetson@192.168.55.1
```

DockerのIDを取得します。

```
sudo docker ps
```

![](./img/docker01.jpg)


Dockerを`trt_pose`という名称で、commitします。

```
sudo docker commit ID(現在起動中のもの)  trt_pose
```

現在起動中のDockerを停止します。

```
sudo docker stop ID(現在起動中のもの)
```
